## 📞 Common Questions

**Q: Where do I edit the content?**
A: Edit text directly in `index.html`. All sections are clearly labeled.

**Q: How do I change the rose color?**
A: Update `--highlight-rgb: 233 0 131;` in `styles.css` to your color's RGB values.

**Q: How do I add images?**
A: Replace the image placeholder divs with `<img>` tags. See DESIGN_GUIDE.md for examples.

**Q: Does this work on mobile?**
A: Yes! Fully responsive. The layout adapts at 640px and 1024px breakpoints.

**Q: Do I need to compile or build anything?**
A: No! It's pure HTML/CSS/JavaScript. Just open in a browser.

**Q: Can I deploy this directly to a server?**
A: Yes! Copy all 3 files and deploy. No build step needed.

**Q: Will it work on older browsers?**
A: Modern browsers only (Chrome, Firefox, Safari, Edge 90+).

**Q: How do I test it locally?**
A: Open `index.html` directly, or start a local server with `python3 -m http.server 8000`.
